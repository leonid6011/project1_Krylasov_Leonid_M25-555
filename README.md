# Игра "Лабиринт сокровищ"

Данный проект является учебным по дисциплине "Основы программирования на Python".
Это текстовая приключенческая игра, в которой игрок исследует лабиринт, решает загадки, сталкивается со случайными событиями и стремится найти сокровище.

## Игровой процесс
Вы начинаете в темном входе лабиринта.
Чтобы пройти игру, необходимо исследовать комнаты, собирать предметы, решать загадки и открыть сундук с сокровищами.

## Установка и запуск:
1. Склонируйте репозиторий:   
```bash
git clone https://github.com/leonid6011/project1_Krylasov_Leonid_M25-555.git
cd project1_Krylasov_Leonid_M25-555
```
2. Установите зависимости:
```bash
make install 
```
или
```bash
poetry install 
```
3. Запустите игру:
```bash
make project 
```
или
```bash
poetry run project 
```

Примеры команд:
```bash
go north # пойти на север
go south # пойти на юг
go east # пойти на восток
go west # пойти на запад
look # осмотреть текущую комнату
take <item> # поднять предмет
use <item> # использовать предмет из инвентаря
inventory # показать инвентарь
solve # попытаться решить загадку в комнате
quit # выйти из игры
help # вывод доступных команд
```

Особенности:
- Загадки с альтернативными ответами
- Реализованы случайные события
- Возможность попасть в ловушку при отсутствии факела
